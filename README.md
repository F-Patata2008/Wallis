# Wallis
Por Felipe Colli, Benjamin Bascuñan, Joaquín Pérez, Agustin Concha, Diego Alvarado

---

## Nuestro Proyecto
### Valores
- Responsabilidad Medioambiental
- Determicnacion
- Compañerismo
- Autosuficiencia

### Tematica
Un playa limpia. Chile tiene casi 6500 kms de costa, y posse aproxiamdamnete 900 playas, de las cuales todas estan contaminadas con plasticos, basura y microplasticos

### Wallis
Nuestro poryecto es crear un robot autonomo, tipo tractor, que fucnione de noche, y atarves de la filtarcion de arena, sea capaz de remover plasticos y basura de las playas chilenas
